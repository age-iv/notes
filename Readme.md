# Notes Service - Веб-сервис для заметок

Полнофункциональный веб-сервис для управления заметками с REST API на Symfony и UI на React.

## Функциональность

- ✅ Создание заметок
- ✅ Просмотр списка заметок
- ✅ Редактирование заметок
- ✅ Удаление заметок
- ✅ Валидация данных
- ✅ Docker-развертывание
- ✅ Swagger документация
- ✅ PHPUnit тесты

## Технологии

### Бэкенд:
- PHP 8.2
- Symfony 7
- MySQL 8.0
- Nginx
- PHPUnit

### Фронтенд:
- React
- Material-UI
- Axios
